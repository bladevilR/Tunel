```powershell

```

默认参数：

| 项 | 值 |
|---|---|
| 监听地址 | `0.0.0.0` |
| 端口 | `8765` |

## 4. 功能范围

已具备：

- 项目录入。
- 站点基础信息、客流、运营、配套数据补齐。
- 互联互通必要性评分。
- 推荐联通方式。
- 规则追溯、缺失项、证据和报告内容生成。
- 项目保存、读取、删除。
- Word、PDF、JSON、CSV 导出。
- 知识库检索。
- 示意图 2D/3D 表达。
- 地上/地下体块表达。
- 建筑层数编辑。
- 多建筑、多通道。
- Windows 本地服务交付包。

最近变更：

- 默认监听地址改为 `0.0.0.0`。
- 站点切换时，系统自动填充字段会同步刷新，人工改过的字段保留。
- 示意图增加地上/地下属性。
- 通道可选择地上/地下。
- 通道支持多条。
- 建筑层数限制：地上 `0-80`，地下 `0-12`。

## 7. 环境变量

| 变量 | 说明 |
|---|---|
| `AMAP_JS_KEY` | 高德地图 JS API Key |
| `AMAP_SECURITY_CODE` | 高德地图安全密钥 |
| `ANTHROPIC_API_KEY` | Claude Code / Anthropic API Key；存在时优先于 `LLM_*` |
| `ANTHROPIC_MODEL` | Claude / Anthropic 模型名称 |
| `ANTHROPIC_BASE_URL` | 可选 Anthropic 服务地址，默认 `https://api.anthropic.com/v1` |
| `LLM_BASE_URL` | 外部模型服务地址 |
| `LLM_API_KEY` | 外部模型 API Key |
| `LLM_MODEL` | 外部模型名称 |
| `SEARCH_API_URL` | 可选搜索服务地址 |
| `SEARCH_API_KEY` | 可选搜索服务 Key |
| `INTERCONNECT_HOST` | 部署包监听地址 |
| `INTERCONNECT_PORT` | 部署包端口 |
| `INTERCONNECT_SKIP_BROWSER` | 启动时不自动打开浏览器 |
| `INTERCONNECT_SKIP_SETUP` | 跳过依赖安装 |
| `INTERCONNECT_NO_PAUSE` | 脚本退出时不暂停 |

## 10. 已知限制

- 当前是本地 MVP，不是正式产品。
- 数据层为 JSON 文件，不支持多人并发和权限审计。
- `frontend/schematic/user_geometry.json` 是全局示意图保存文件，未按项目拆分。
- 示意图是表达工具，不是 CAD/BIM/GIS 工程模型。
- 地下空间只做图面表达和基础属性，不含工程可行性校验。
- 导出文件较多时需要清理 `exports/`。
- 高德地图、外部模型、搜索服务依赖密钥和网络。
- 局域网访问需服务器 IP 和防火墙端口放行。
